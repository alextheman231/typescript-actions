## Description of Changes

<!-- user-editable-section-start -->
- Log the installed version at the end of `safe-npm-dependency-global-install`
    - This makes it easier to debug in cases where the action may've ended up resolving the version range incorrectly for whatever reason.
    - If it does, we'll at least be able to clearly see in the logs exactly what sort of mismatch may have occurred.
