This is a new minor release of the `typescript-actions` package. It introduces new features and other backwards-compatible changes that should require little to no refactoring. Please read the description of changes below.
<!-- alex-c-line-release-summary-end -->

## Description of Changes

<!-- user-editable-section-start -->
- Change the workflow inputs to be `kebab-case` rather than `snake_case`.
    - I think `kebab-case` is generally preferred in actions. It is what tends to be used more officially in most official actions.
    - Technically this should be a major release, but it is also early days and it only really needs to be refactored in one place in my setup as of now, at least.
    - Changes like this will be versioned major going forward, however.
